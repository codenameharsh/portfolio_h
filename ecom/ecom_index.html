E-Commerce Interface

User Interface Design Front-End Development Meta Front-End Capstone Project

PROJECT OVERVIEW

From Coursework To Commerce

I used my Meta capstone as a springboard to tackle a bigger challenge: building a functional e-commerce site from the ground up.

I spent as much time refining the JavaScript logic as I did the typography.

I wanted to prove that you don't need a heavy framework to create a smooth, frictionless experience that handles real-time updates and complex user flows.

CHALLENGES AND TAKEAWAYS

THE HURDLES

  • Syncing data with the UI was a bigger challenge than the CSS.
  • Building from scratch showed me why frameworks like React matter.
  • I chose reusable functions over quick, messy fixes.
  • Handling instant updates was a steep but rewarding curve.

THE INSIGHTS

  1. If a user has to think about the UI, I haven't done my job.
  2. Front-end is a design challenge—code is just the tool to solve it.
  3. On mobile, one clear button beats a dozen fancy features.
  4. The goal is to make technical complexity feel invisible.

impact & links

FINAL OUTCOME

A production-ready prototype that proves you don’t need a framework to build a seamless user experience.
